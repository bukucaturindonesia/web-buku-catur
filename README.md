## Cara Hosting Gratis di GitHub Pages

1. Buat repository baru di GitHub.
2. Upload semua file dari folder ini ke repository tersebut.
3. Buka menu **Settings** di repository.
4. Masuk ke **Pages**.
5. Pada bagian **Build and deployment**, pilih branch `main` dan folder `/root`.
6. Klik **Save**.
7. Tunggu beberapa saat sampai GitHub memberikan link website.
